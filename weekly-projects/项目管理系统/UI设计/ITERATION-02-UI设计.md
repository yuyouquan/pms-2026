### 1.2 甘特图样式

- 进度条颜色：未开始(灰色)、进行中(蓝色)、已完成(绿色)
- 时间刻度：按天显示
- 周末背景：浅灰色底

## 2. 拖动排序

### 2.1 交互状态

```
序号左侧图标：
- 默认：☰ (三横线，表示可拖动)
- 拖动中：📍 (定位图标)
- 禁用：不可拖动（子任务继承父任务）
```

### 2.2 拖动反馈

- 拖动时显示插入线
- 释放后序号自动更新
- 子任务跟随父任务移动

### 5.1 搜索框

- 位置：操作栏右侧
