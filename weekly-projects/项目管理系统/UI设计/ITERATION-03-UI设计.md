## 1. 拖动排序

### 1.1 交互样式

```
拖动前：
┌────┬─────────┬──────┐
│ 1  │ 概念    │ ☰    │
├────┼─────────┼──────┤
│ 2  │ 计划    │ ☰    │
├────┼─────────┼──────┤
│ 3  │ 开发验证 │ ☰    │
└────┴─────────┴──────┘

拖动中（2拖到1和3之间）：
┌────┬─────────┬──────┐
│ 1  │ 概念    │ ☰    │
├────┼─────────┼──────┤
│    │ ═══════ │ ← 插入指示线
├────┼─────────┼──────┤
│ 2  │ 计划    │ ☰    │
└────┴─────────┴──────┘

拖动后（序号自动更新）：
┌────┬─────────┬──────┐
│ 1  │ 概念    │ ☰    │
├────┼─────────┼──────┤
│ 2  │ 开发验证 │ ☰    │ ← 原3变成2
├────┼─────────┼──────┤
│ 3  │ 计划    │ ☰    │ ← 原2变成3
└────┴─────────┴──────┘
```

### 1.2 层级拖动

```
层级提升（1.1拖到1旁边）：
┌────┬─────────┐
│ 1  │ 概念    │ ☰
├────┼─────────┤
│    │ ↑ 提升  │ ← 释放后1.1变成2
└────┴─────────┘

层级下降（2拖到1.1下）：
┌────┬─────────┐
│ 1  │ 概念    │ ☰
├────┼─────────┤
│ 1.1│ 概念启动│ ☰
├────┼─────────┤
│    │ ↓ 下降  │ ← 释放后2变成2.1
└────┴─────────┘
```
