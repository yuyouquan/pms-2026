# UI原型设计文档 - ITERATION-01

**迭代版本**: ITERATION-01  
**设计模块**: 配置中心 - 一级计划模板管理 & 二级计划模板管理

## 1. 页面布局

### 1.1 整体布局

```
┌─────────────────────────────────────────────────────────┐
│  顶部导航栏 (64px)                                       │
│  [Logo] [工作台] [配置中心] [用户信息]                    │
├─────────────────────────────────────────────────────────┤
│  侧边栏 (240px)  │  主内容区                             │
│  - 配置中心      │  ┌─────────────────────────────────┐ │
│    - 一级计划模板│  │  页面标题 + 面包屑               │ │
│    - 二级计划模板│  ├─────────────────────────────────┤ │
│                  │  │  筛选条件区                      │ │
│                  │  │  [项目类型下拉] [版本下拉]        │ │
│                  │  ├─────────────────────────────────┤ │
│                  │  │  操作按钮区                      │ │
│                  │  │  [创建修订] [编辑] [发布] [版本对比]│ │
│                  │  ├─────────────────────────────────┤ │
│                  │  │  计划表格/甘特图                  │ │
│                  │  │  ...                            │ │
│                  │  └─────────────────────────────────┘ │
└─────────────────────────────────────────────────────────┘
```

### 1.2 页面宽度
- 最大宽度: 1440px
- 侧边栏宽度: 240px
- 内容区内边距: 24px

### 2.6 拖拽与添加子项


## 5. 交互规范

### 5.1 按钮状态
- 默认: 背景色正常
- 悬停: 背景色加深 10%
- 禁用: 背景色 50% 透明度，cursor: not-allowed

### 5.2 删除确认
- 首次点击: 无反应
- 二次点击: 弹出确认弹窗
- 弹窗内容: "是否确认删除？删除后不可恢复"
- 确认按钮: 红色 #EF4444

### 5.3 版本切换
- 切换动画: 200ms 淡入淡出
- 加载状态: 骨架屏显示
