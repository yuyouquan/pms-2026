## 1. 问题提交指南

### 1.1 问题分类

| 类型 | 标签 | 描述 |
|------|------|------|
| 功能缺陷 | bug | 功能不正常 |
| 功能请求 | feature | 需求建议 |
| 界面问题 | ui | 样式/交互问题 |
| 性能问题 | performance | 加载慢、卡顿 |
| 安全问题 | security | 漏洞风险 |

---

## 2. 提交格式

请使用以下模板提交Issue：

```markdown
## 问题描述
[简洁描述问题]

## 复现步骤
1. 步骤1
2. 步骤2

## 预期行为
[期望的结果]

## 实际行为
[实际的结果]

## 环境信息
- 浏览器：
- 操作系统：
- 版本：

## 截图/日志
[如有则添加]
```
