## 1. 工作台-新增/编辑项目

### 1.1 新增项目按钮

```
+----------------------------------------------------------+
|  项目卡片  |  项目看板  |  代办中心     [+ 新增项目]     |
+----------------------------------------------------------+
```

### 1.2 新增/编辑弹窗

```
+----------------------------------------------------------+
|  新增项目 / 编辑项目                              [X]    |
+----------------------------------------------------------+
|  项目名: [________________]                               |
|  项目类型: [整机产品项目 ▼]                              |
|  安卓版本: [Android 16 ▼]                                |
|  芯片平台: [MTK ▼]                                      |
|  项目状态: [筹备中 ▼]                                    |
|  SPM: [张三 ▼]                                          |
|  (如果是整机产品项目)                                    |
|  市场: [OP] [TR] [RU] (可多选)                          |
|                                                        |
|  [取消] [保存]                                          |
+----------------------------------------------------------+
```

## 3. 二级计划创建

```
+----------------------------------------------------------+
|  ...  [创建二级计划]                                     |
+----------------------------------------------------------+

+----------------------------------------------------------+
|  创建二级计划                                      [X]   |
+----------------------------------------------------------+
|  绑定里程碑(多选): [STR1, STR2, STR3, ...]             |
|  计划模板类型: [需求开发计划 ▼]                          |
|  二级计划名称: [需求开发计划] (自动填不可改)             |

|  [取消] [创建]                                          |
+----------------------------------------------------------+
```
